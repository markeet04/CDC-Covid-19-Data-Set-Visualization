About This Project

This COVID-19 data visualization project is built using publicly available data from the CDC. The data includes information on total deaths by state, age group, and gender, and compares COVID-19 deaths to other causes. Due to API request limits, the project is currently limited to a subset of states.

The visualizations are created using D3.js, a powerful JavaScript library for producing dynamic, interactive data visualizations in web browsers. The project demonstrates how such visualizations can be developed and expanded to include more data, states, or additional metrics as needed.

Future improvements could include integrating real-time data updates, adding more states and data points, and further optimizing the visualizations for performance and clarity.
